## 当前实验状态（2026-07-23 深夜）

- 已实现：两阶段回合制（04）+ **两段式玩家掷骰**（01 落地：裁决发起→玩家
  点掷骰→服务端权威骰值→结算叙事，链式检定自然涌现）+ 长度纪律（历史截断
  /max_tokens 兜底）+ 迷茫引导规则。

  奖励、求引导轮真在引导（169 字指向酒瓶把柄路线）。
