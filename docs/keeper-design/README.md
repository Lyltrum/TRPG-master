## 当前实验状态（2026-07-23）

- 已实现：单段式（agent 一次回应内部掷完，结果写进叙事）。
- 已验证：《追书人》五轮真实 DeepSeek 冒烟跑通，掷骰路径实测正常。
- 未实现：两段式（玩家亲手摇骰）、线下模式、第二个结构不同的模组。
- 本轮重点观测：B（agent 的检定发起判定准不准）——这是决定要不要投入 A 的地基数据。
