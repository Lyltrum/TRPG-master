## 当前实验状态（2026-07-23 · v2）

- 已实现：两阶段回合制（裁决 JSON → 代码执行 → 纯叙事），检定发起从"模型
  自觉"变成"schema 必填"；掷骰可见性由代码硬保证。
- 已验证：隐式动作（不喊技能名）正确触发剧本标注的检定；状态每轮记账；
  Sonnet 模拟玩家的考察矩阵中 v1 唯一失守项（主动发起检定）已翻绿。
- 未实现：时钟/议程（已拍板下一版）、场景指针结构化、线索账本、两段式
  玩家亲手摇骰、第二个结构不同的模组。
